Tools Used To Make This

There are two major tools used to make everything you read here: Markdown for the writing, and Caddy for compiling it.

Markdown is a form of markup designed by John Gruber and Aaron Swartz. The advantages of Markdown for writing is that it is a plain text format that makes converting to other formats very simple. There are many tools for converting Markdown to HTML and PDF, where the final output can be more easily stylized. And since it is stored in plaintext, it can be created and edited in any number of text editors. Currently, I switch between iA Writer and Sublime Text for all Markdown writing.

All of these Markdown files are then served through Caddy, a new web server written in Go. Caddy has built-in support for rendering Markdown files to HTML using a basic template system. While it is not as powerful as Ghost or Hugo, it fulfills my needs, and I would be using Caddy even without the Markdown support as it supports negotiating SSL certificates through Let’s Encrypt automatically.
